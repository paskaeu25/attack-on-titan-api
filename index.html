Attack On Titan API

URLs:

  1. Base URL: [enter URL]

  2. Get list of all characters: GET /api/characters

  3. Get character by Id: GET /api/characters/{id}

  4. Delete character: DELETE /api/characters/{id}

  5. Create new character: POST /api/characters

    • Request Header:

      Content-Type: application/json

    • Request Body:

      {
      "name": "Char name",
      "affiliation": "affiliation name",
      "titan": "name of titan",
      "img": "url to image",
      "gif": "url to gif"
      }

  6. Update character: PUT /api/characters/{id}

    • Request Header:

      Content-Type: application/json

    • Request Body:

      {
      "name": "New Char name",
      "affiliation": "New affiliation name",
      "titan": "new name of titan",
      "img": "new url to image",
      "gif": "new url to gif"
      }
